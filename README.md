# Anticancer_peptide
Predict Anticancer peptide  
核心设计理念
本项目遵循"多尺度表征学习 + 物理化学先验知识 + 自适应优化"的三位一体设计理念：
1.多模态特征融合：整合序列特征、结构特征、进化特征和深度语义特征
2.端到端可训练：突破传统"特征提取+分类器"的割裂模式，实现Transformer编码器的任务自适应微调
3.梯度方向一致性优化：创新提出基于梯度方向稳定性的自适应学习率调整机制,解决深度学习在生物序列数据上的训练震荡问题  
  
本项目基于多种开源技术栈构建，充分利用现有开源生态实现快速开发与功能扩展：
Gradio：用于快速构建机器学习模型的Web交互界面，实现前后端集成
PyTorch：用于深度学习模型的构建与训练，支持高效的张量计算与模型部署
Plotly：用于实现预测结果的可视化展示（如置信度仪表盘）
NumPy / Pandas：用于数据预处理、批量数据分析与结果统计
